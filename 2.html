CSS

CSS ( Cascading Style Sheets )는 HTML 또는 XML ( SVG , MathML 또는 XHTML 과 같은 XML 방언 포함 ) 과 같은 마크업 언어로 작성된 문서 의 표시 를 설명하는 데 사용되는 스타일시트 언어 입니다. [1] CSS는 HTML 및 JavaScript 와 함께 World Wide Web 의 초석 기술입니다 . [2] CSS는 레이아웃 , 색상 및 글꼴 을 포함하여 내용과 프레젠테이션을 분리 할 수 있도록 설계되었습니다 . [3] 이러한 분리는 콘텐츠 접근성 을 향상시킬 수 있습니다 . 프레젠테이션 특성의 사양에 더 많은 유연성과 제어 기능을 제공합니다. 별도의 .css 파일에 관련 CSS를 지정하여 여러 웹 페이지 에서 형식을 공유할 수 있도록 하여 구조적 콘텐츠의 복잡성과 반복을 줄입니다. .css 파일을 캐시 하여 파일을 공유하는 페이지와 해당 형식 사이의 페이지 로드 속도를 개선합니다. 서식과 콘텐츠를 분리하면 화면, 인쇄, 음성(음성 기반 브라우저 또는 화면 판독기 사용 ) 및 점자 기반 과 같은 다양한 렌더링 방법에 대해 서로 다른 스타일로 동일한 마크업 페이지를 표시할 수 있습니다. 촉각 장치. 또한 CSS에는 콘텐츠가 모바일 장치 에서 액세스되는 경우 대체 형식 지정에 대한 규칙이 있습니다 . [4] 캐스케이딩 이라는 이름 은 특정 요소와 일치하는 규칙이 두 개 이상인 경우 적용할 스타일 규칙을 결정하기 위해 지정된 우선 순위 체계에서 가져옵니다. 이 계단식 우선 순위 체계는 예측 가능합니다. CSS 사양은 W3C( World Wide Web Consortium )에서 관리합니다. 인터넷 미디어 유형( MIME 유형 ) text/css은 RFC 2318(1998년 3월)에 의해 CSS와 함께 사용하도록 등록되었습니다. W3C는 CSS 문서에 대한 무료 CSS 유효성 검사 서비스 를 운영합니다. [5]
